# `lazygit` asdf plugin

This is the `lazygit` integration for the [universal-asdf-plugin](../../) project.

[lazygit](https://github.com/jesseduffield/lazygit) is a simple terminal UI for Git commands that presents repositories in an interactive interface. It speeds up common Git workflows like staging, committing, branching, and resolving conflicts without leaving the terminal.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all lazygit

# Install latest version
universal-asdf-plugin install lazygit latest
```
