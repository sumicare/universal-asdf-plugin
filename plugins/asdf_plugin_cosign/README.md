# `cosign` asdf plugin

This is the `cosign` integration for the [universal-asdf-plugin](../../) project.

[cosign](https://github.com/sigstore/cosign) is a tool for signing, verifying, and storing container images and related artifacts. It supports keyless signatures, attestations, and integration with transparency logs to help secure software supply chains.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all cosign

# Install latest version
universal-asdf-plugin install cosign latest
```
