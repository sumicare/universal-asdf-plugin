# `python` asdf plugin

This is the `python` integration for the [universal-asdf-plugin](../../) project.

[Python](https://www.python.org/) is a high-level, general-purpose programming language known for its readability and rich ecosystem of libraries. It is widely used for web development, scripting, data science, automation, and many other domains.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all python

# Install latest version
universal-asdf-plugin install python latest
```
