# `sops` asdf plugin

This is the `sops` integration for the [universal-asdf-plugin](../../) project.

sops is an editor and CLI tool for managing encrypted files. It supports multiple key management backends and lets you keep configuration files encrypted at rest while still being easy to edit.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all sops

# Install latest version
universal-asdf-plugin install sops latest
```
