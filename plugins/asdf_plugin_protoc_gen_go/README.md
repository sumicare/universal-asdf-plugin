# `protoc-gen-go` asdf plugin

This is the `protoc-gen-go` integration for the [universal-asdf-plugin](../../) project.

[protoc-gen-go](https://github.com/protocolbuffers/protobuf-go) is the Go code generator plugin for Protocol Buffers. It works with protoc to generate Go types and helper methods from .proto files so Go applications can easily work with protobuf messages.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all protoc-gen-go

# Install latest version
universal-asdf-plugin install protoc-gen-go latest
```
