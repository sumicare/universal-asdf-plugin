# `grype` asdf plugin

This is the `grype` integration for the [universal-asdf-plugin](../../) project.

[Grype](https://github.com/anchore/grype) is a vulnerability scanner for container images and filesystems. It detects known vulnerabilities in packages and OS components by comparing software inventories against public vulnerability databases.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all grype

# Install latest version
universal-asdf-plugin install grype latest
```
