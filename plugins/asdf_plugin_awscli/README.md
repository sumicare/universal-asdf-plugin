# `awscli` asdf plugin

This is the `awscli` integration for the [universal-asdf-plugin](../../) project.

[`aws` CLI](https://aws.amazon.com/cli/) is the unified command line interface for Amazon Web Services, providing scripted access to nearly every AWS API. It lets you manage infrastructure, deploy applications, and automate workflows from the terminal or CI/CD pipelines.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all awscli

# Install latest version
universal-asdf-plugin install awscli latest
```
