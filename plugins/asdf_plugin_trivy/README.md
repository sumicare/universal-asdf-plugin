# `trivy` asdf plugin

This is the `trivy` integration for the [universal-asdf-plugin](../../) project.

Trivy is a comprehensive vulnerability and misconfiguration scanner for containers, file systems, and other artifacts. It checks for CVEs, insecure configurations, and exposed secrets across images, source code, and infrastructure-as-code.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all trivy

# Install latest version
universal-asdf-plugin install trivy latest
```
