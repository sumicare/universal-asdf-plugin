# `tfupdate` asdf plugin

This is the `tfupdate` integration for the [universal-asdf-plugin](../../) project.

tfupdate is a tool for updating Terraform module and provider versions.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all tfupdate

# Install latest version
universal-asdf-plugin install tfupdate latest
```
