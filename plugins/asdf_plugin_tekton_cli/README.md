# `tekton-cli` asdf plugin

This is the `tekton-cli` integration for the [universal-asdf-plugin](../../) project.

Tekton CLI provides a command-line interface for interacting with Tekton Pipelines on Kubernetes. It helps you manage pipelines, tasks, and runs from the terminal, making it easier to debug and monitor CI/CD workloads.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all tekton-cli

# Install latest version
universal-asdf-plugin install tekton-cli latest
```
