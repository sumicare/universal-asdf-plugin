# `github-cli` asdf plugin

This is the `github-cli` integration for the [universal-asdf-plugin](../../) project.

[GitHub CLI](https://cli.github.com/) (gh) is the official command-line interface for working with GitHub from your terminal. It helps you manage repositories, issues, pull requests, releases, and workflows without leaving the command line.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all github-cli

# Install latest version
universal-asdf-plugin install github-cli latest
```
