# `terragrunt` asdf plugin

This is the `terragrunt` integration for the [universal-asdf-plugin](../../) project.

Terragrunt is a thin wrapper for Terraform that provides extra tools for keeping configurations DRY and manageable. It adds features like remote state configuration, dependency management, and reusable modules to simplify large Terraform codebases.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all terragrunt

# Install latest version
universal-asdf-plugin install terragrunt latest
```
