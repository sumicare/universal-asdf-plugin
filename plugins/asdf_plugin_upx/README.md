# `upx` asdf plugin

This is the `upx` integration for the [universal-asdf-plugin](../../) project.

UPX is a packer for executables that compresses binaries to reduce their size on disk. It can compress and decompress programs quickly, often with little to no impact on runtime performance.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all upx

# Install latest version
universal-asdf-plugin install upx latest
```
