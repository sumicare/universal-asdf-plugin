# `pipx` asdf plugin

This is the `pipx` integration for the [universal-asdf-plugin](../../) project.

[pipx](https://pipx.pypa.io/) is a tool for installing and running isolated Python applications in dedicated virtual environments. It makes it easy to run and upgrade Python CLIs globally without polluting your main Python environment.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all pipx

# Install latest version
universal-asdf-plugin install pipx latest
```
