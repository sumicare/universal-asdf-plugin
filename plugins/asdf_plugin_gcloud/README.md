# `gcloud` asdf plugin

This is the `gcloud` integration for the [universal-asdf-plugin](../../) project.

[Google Cloud SDK](https://cloud.google.com/sdk) (gcloud) is the command-line interface for managing Google Cloud Platform resources and services. It lets you authenticate, configure projects, and interact with compute, storage, networking, and higher-level APIs directly from your terminal or automation.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all gcloud

# Install latest version
universal-asdf-plugin install gcloud latest
```
