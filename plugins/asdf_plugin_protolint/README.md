# `protolint` asdf plugin

This is the `protolint` integration for the [universal-asdf-plugin](../../) project.

[protolint](https://github.com/yoheimuta/protolint) is a linter for Protocol Buffers definitions. It enforces style and best practices in .proto files, helping teams keep protobuf schemas consistent and readable across repositories.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all protolint

# Install latest version
universal-asdf-plugin install protolint latest
```
