# `shfmt` asdf plugin

This is the `shfmt` integration for the [universal-asdf-plugin](../../) project.

shfmt is a shell script formatter.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all shfmt

# Install latest version
universal-asdf-plugin install shfmt latest
```
