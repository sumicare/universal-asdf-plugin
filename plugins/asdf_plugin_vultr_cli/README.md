# `vultr-cli` asdf plugin

This is the `vultr-cli` integration for the [universal-asdf-plugin](../../) project.

vultr-cli is the official command-line interface for Vultr cloud services. It allows you to manage instances, networking, DNS, and other Vultr resources programmatically from your terminal or automation.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all vultr-cli

# Install latest version
universal-asdf-plugin install vultr-cli latest
```
