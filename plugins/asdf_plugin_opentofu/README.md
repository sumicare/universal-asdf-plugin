# `opentofu` asdf plugin

This is the `opentofu` integration for the [universal-asdf-plugin](../../) project.

[OpenTofu](https://opentofu.org/) is an open source fork of Terraform for infrastructure as code. It provides a community-driven alternative for defining, provisioning, and managing cloud and on-premises resources declaratively.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all opentofu

# Install latest version
universal-asdf-plugin install opentofu latest
```
