# `nodejs` asdf plugin

This is the `nodejs` integration for the [universal-asdf-plugin](../../) project.

[Node.js](https://nodejs.org/) is a JavaScript runtime built on Chrome's V8 engine that enables server-side and command-line JavaScript applications. It powers modern web backends, tooling, and APIs with a non-blocking I/O model and a vast npm package ecosystem.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all nodejs

# Install latest version
universal-asdf-plugin install nodejs latest
```
