# `sccache` asdf plugin

This is the `sccache` integration for the [universal-asdf-plugin](../../) project.

[sccache](https://github.com/mozilla/sccache) is a shared compilation cache for speeding up builds by reusing results from previous compilations. It can be used locally or as a distributed cache to reduce build times across developers and CI systems.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all sccache

# Install latest version
universal-asdf-plugin install sccache latest
```
