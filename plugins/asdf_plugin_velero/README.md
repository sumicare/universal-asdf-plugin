# `velero` asdf plugin

This is the `velero` integration for the [universal-asdf-plugin](../../) project.

Velero is a tool for backing up and restoring Kubernetes cluster resources and volumes. It supports scheduled backups, disaster recovery, and cluster migrations by capturing both resource manifests and persistent data.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all velero

# Install latest version
universal-asdf-plugin install velero latest
```
