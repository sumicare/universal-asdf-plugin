# `k9s` asdf plugin

This is the `k9s` integration for the [universal-asdf-plugin](../../) project.

[k9s](https://k9scli.io/) is a terminal UI for managing and observing Kubernetes clusters, providing an interactive view over your workloads and resources. It streamlines common kubectl operations and lets you drill into logs, events, and configuration from an always-on TUI.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all k9s

# Install latest version
universal-asdf-plugin install k9s latest
```
