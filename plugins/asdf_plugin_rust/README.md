# `rust` asdf plugin

This is the `rust` integration for the [universal-asdf-plugin](../../) project.

[Rust](https://www.rust-lang.org/) is a systems programming language focused on safety, concurrency, and performance without a garbage collector. Its ownership model helps prevent data races and memory bugs while still enabling low-level control and high-level expressiveness.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all rust

# Install latest version
universal-asdf-plugin install rust latest
```
