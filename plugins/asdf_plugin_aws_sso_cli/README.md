# `aws-sso-cli` asdf plugin

This is the `aws-sso-cli` integration for the [universal-asdf-plugin](../../) project.

[aws-sso-cli](https://github.com/synfinatic/aws-sso-cli) is a command-line helper for AWS Single Sign-On (SSO) authentication. It streamlines logging in via AWS SSO and exports credentials or config entries so you can use AWS CLI and SDKs with SSO-backed accounts.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all aws-sso-cli

# Install latest version
universal-asdf-plugin install aws-sso-cli latest
```
