# `asdf` asdf plugin

This is the `asdf` integration for the [universal-asdf-plugin](../../) project.

[asdf](https://asdf-vm.com/) is a tool version manager that lets you manage multiple runtimes and CLIs with a single interface. It supports per-project and global versions using `.tool-versions` files so you can keep development environments consistent across machines and teams.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all asdf

# Install latest version
universal-asdf-plugin install asdf latest
```
