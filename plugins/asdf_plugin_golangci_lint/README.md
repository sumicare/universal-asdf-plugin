# `golangci-lint` asdf plugin

This is the `golangci-lint` integration for the [universal-asdf-plugin](../../) project.

[golangci-lint](https://golangci-lint.run/) is a fast Go linters aggregator that runs many linters in parallel. It helps enforce code quality and style by combining multiple linters into a single, configurable tool that integrates well with editors and CI.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all golangci-lint

# Install latest version
universal-asdf-plugin install golangci-lint latest
```
