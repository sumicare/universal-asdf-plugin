# `tflint` asdf plugin

This is the `tflint` integration for the [universal-asdf-plugin](../../) project.

TFLint is a Terraform linter that detects errors and enforces best practices in Terraform configurations. It supports built-in and plugin-based rules to catch potential issues before they reach your infrastructure.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all tflint

# Install latest version
universal-asdf-plugin install tflint latest
```
