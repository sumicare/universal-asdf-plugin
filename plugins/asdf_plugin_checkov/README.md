# `checkov` asdf plugin

This is the `checkov` integration for the [universal-asdf-plugin](../../) project.

[Checkov](https://github.com/bridgecrewio/checkov) is a static code analysis tool for infrastructure-as-code that scans Terraform, CloudFormation, Kubernetes manifests, and more for misconfigurations. It provides built-in policies and custom rules so you can catch security and compliance issues before changes reach production.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all checkov

# Install latest version
universal-asdf-plugin install checkov latest
```
