# `nerdctl` asdf plugin

This is the `nerdctl` integration for the [universal-asdf-plugin](../../) project.

[nerdctl](https://github.com/containerd/nerdctl) is a Docker-compatible CLI for containerd that brings familiar container commands to the containerd runtime. It supports images, volumes, networks, and advanced features like rootless containers and cgroups v2.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all nerdctl

# Install latest version
universal-asdf-plugin install nerdctl latest
```
