# `shellcheck` asdf plugin

This is the `shellcheck` integration for the [universal-asdf-plugin](../../) project.

[ShellCheck](https://www.shellcheck.net/) is a static analysis tool for shell scripts that detects common pitfalls, bugs, and style issues. It provides detailed suggestions and explanations so you can write more robust and portable shell code.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all shellcheck

# Install latest version
universal-asdf-plugin install shellcheck latest
```
