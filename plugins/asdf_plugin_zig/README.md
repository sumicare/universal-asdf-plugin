# `zig` asdf plugin

This is the `zig` integration for the [universal-asdf-plugin](../../) project.

Zig is a general-purpose programming language and toolchain for robust, optimal software with explicit control over memory and performance. It focuses on predictability, simplicity, and interoperability with C for systems programming use cases.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all zig

# Install latest version
universal-asdf-plugin install zig latest
```
