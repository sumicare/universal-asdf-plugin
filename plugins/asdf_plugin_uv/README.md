# `uv` asdf plugin

This is the `uv` integration for the [universal-asdf-plugin](../../) project.

[uv](https://docs.astral.sh/uv/) is a fast Python package installer and environment manager designed to make creating, syncing, and reproducing environments lightweight. It focuses on performance and reproducibility so teams can share exact Python setups across machines and CI systems.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all uv

# Install latest version
universal-asdf-plugin install uv latest
```
