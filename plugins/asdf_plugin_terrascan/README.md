# `terrascan` asdf plugin

This is the `terrascan` integration for the [universal-asdf-plugin](../../) project.

Terrascan is a static code analyzer for detecting compliance and security violations in IaC.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all terrascan

# Install latest version
universal-asdf-plugin install terrascan latest
```
