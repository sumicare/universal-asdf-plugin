# `ginkgo` asdf plugin

This is the `ginkgo` integration for the [universal-asdf-plugin](../../) project.

[Ginkgo](https://github.com/onsi/ginkgo) is a behavior-driven development (BDD) testing framework for Go. It provides an expressive DSL for structuring tests and integrates closely with the Gomega matcher library to write readable, robust test suites.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all ginkgo

# Install latest version
universal-asdf-plugin install ginkgo latest
```
