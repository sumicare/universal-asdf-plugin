# `ko` asdf plugin

This is the `ko` integration for the [universal-asdf-plugin](../../) project.

[ko](https://ko.build/) is a container image builder for Go applications that eliminates the need for Dockerfiles. It builds images directly from Go source and pushes them to container registries, integrating smoothly with Kubernetes workflows.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all ko

# Install latest version
universal-asdf-plugin install ko latest
```
