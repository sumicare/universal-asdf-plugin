# `kind` asdf plugin

This is the `kind` integration for the [universal-asdf-plugin](../../) project.

[kind](https://kind.sigs.k8s.io/) (Kubernetes in Docker) runs local Kubernetes clusters using Docker containers as nodes. It is commonly used for development, testing, and CI environments where a lightweight Kubernetes cluster is needed.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all kind

# Install latest version
universal-asdf-plugin install kind latest
```
