# `goreleaser` asdf plugin

This is the `goreleaser` integration for the [universal-asdf-plugin](../../) project.

[GoReleaser](https://goreleaser.com/) is a release automation tool for building and publishing Go projects. It automates building binaries, creating archives and container images, and publishing releases to package managers and hosting services.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all goreleaser

# Install latest version
universal-asdf-plugin install goreleaser latest
```
