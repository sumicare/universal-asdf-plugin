# `telepresence` asdf plugin

This is the `telepresence` integration for the [universal-asdf-plugin](../../) project.

Telepresence is a local development tool for interacting with remote Kubernetes clusters. It lets you run services locally while connecting them into a remote cluster, enabling realistic debugging and testing without deploying every change.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all telepresence

# Install latest version
universal-asdf-plugin install telepresence latest
```
