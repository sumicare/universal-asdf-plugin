# `doctl` asdf plugin

This is the `doctl` integration for the [universal-asdf-plugin](../../) project.

[doctl](https://github.com/digitalocean/doctl) is the official command-line interface for managing DigitalOcean resources. It lets you provision and manage droplets, Kubernetes clusters, databases, networking, and other platform services from scripts and terminals.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all doctl

# Install latest version
universal-asdf-plugin install doctl latest
```
