# `protoc-gen-go-grpc` asdf plugin

This is the `protoc-gen-go-grpc` integration for the [universal-asdf-plugin](../../) project.

[protoc-gen-go-grpc](https://github.com/grpc/grpc-go/tree/master/cmd/protoc-gen-go-grpc) is the Go code generator plugin for gRPC services. It generates Go service interfaces and client stubs from your protobuf service definitions to help you build type-safe gRPC servers and clients.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all protoc-gen-go-grpc

# Install latest version
universal-asdf-plugin install protoc-gen-go-grpc latest
```
