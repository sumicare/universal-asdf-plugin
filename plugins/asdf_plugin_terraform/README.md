# `terraform` asdf plugin

This is the `terraform` integration for the [universal-asdf-plugin](../../) project.

[Terraform](https://www.terraform.io/) is an infrastructure-as-code tool for provisioning and managing cloud and on-premises resources using declarative configuration files. It plans and applies changes safely, enabling repeatable environments, versioned infrastructure, and collaborative workflows.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all terraform

# Install latest version
universal-asdf-plugin install terraform latest
```
