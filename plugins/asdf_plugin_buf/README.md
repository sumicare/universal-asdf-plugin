# `buf` asdf plugin

This is the `buf` integration for the [universal-asdf-plugin](../../) project.

[Buf](https://buf.build/) is a toolchain for working with Protocol Buffers that improves compilation, linting, breaking-change detection, and publishing. It helps teams standardize proto practices across repositories and automate schema evolution safely.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all buf

# Install latest version
universal-asdf-plugin install buf latest
```
