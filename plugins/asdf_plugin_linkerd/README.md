# `linkerd` asdf plugin

This is the `linkerd` integration for the [universal-asdf-plugin](../../) project.

[Linkerd](https://linkerd.io/) is an ultralight, security-first service mesh for Kubernetes. It adds features like mTLS, traffic shifting, retries, and observability between services without requiring changes to application code.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all linkerd

# Install latest version
universal-asdf-plugin install linkerd latest
```
