# `kubectl` asdf plugin

This is the `kubectl` integration for the [universal-asdf-plugin](../../) project.

[kubectl](https://kubernetes.io/docs/reference/kubectl/) is the command-line interface for interacting with Kubernetes clusters and managing their resources. It provides commands to inspect cluster state, deploy and update workloads, debug issues, and perform day-to-day operations against the Kubernetes API.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all kubectl

# Install latest version
universal-asdf-plugin install kubectl latest
```
