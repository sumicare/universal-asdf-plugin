# `yq` asdf plugin

This is the `yq` integration for the [universal-asdf-plugin](../../) project.

yq is a portable command-line YAML, JSON and XML processor. It lets you query, merge, and transform structured documents in a jq-like way directly from shell scripts and pipelines.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all yq

# Install latest version
universal-asdf-plugin install yq latest
```
