# `sqlc` asdf plugin

This is the `sqlc` integration for the [universal-asdf-plugin](../../) project.

sqlc generates type-safe code from SQL queries. It turns SQL files into idiomatic code in languages like Go so you can use your database schema directly without writing manual query boilerplate.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all sqlc

# Install latest version
universal-asdf-plugin install sqlc latest
```
