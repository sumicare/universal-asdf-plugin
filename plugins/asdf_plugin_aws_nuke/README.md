# `aws-nuke` asdf plugin

This is the `aws-nuke` integration for the [universal-asdf-plugin](../../) project.

[aws-nuke](https://github.com/ekristen/aws-nuke) is a tool for cleaning up and deleting all resources in an AWS account. It is typically used to reset or dispose of test and sandbox environments by enumerating resources across regions and removing them according to configurable filters.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all aws-nuke

# Install latest version
universal-asdf-plugin install aws-nuke latest
```
