# `argocd` asdf plugin

This is the `argocd` integration for the [universal-asdf-plugin](../../) project.

[Argo CD](https://github.com/argoproj/argo-cd) is a declarative, GitOps continuous delivery tool for Kubernetes that continuously reconciles your live cluster state with what is defined in Git. It automates application deployments, rollbacks, and health checks, giving you a clear view of drift and history for each environment.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all argocd

# Install latest version
universal-asdf-plugin install argocd latest
```
