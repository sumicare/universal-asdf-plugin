# `syft` asdf plugin

This is the `syft` integration for the [universal-asdf-plugin](../../) project.

Syft is a CLI tool for generating Software Bill of Materials (SBOMs) from container images and filesystems. It catalogs packages and dependencies so you can understand what software is present for compliance, security, and inventory use cases.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all syft

# Install latest version
universal-asdf-plugin install syft latest
```
