# `jq` asdf plugin

This is the `jq` integration for the [universal-asdf-plugin](../../) project.

[jq](https://jqlang.github.io/jq/) is a lightweight and flexible command-line JSON processor for filtering, transforming, and querying JSON data. It provides a powerful expression language so you can slice, map, and restructure JSON directly in your shell scripts and pipelines.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all jq

# Install latest version
universal-asdf-plugin install jq latest
```
