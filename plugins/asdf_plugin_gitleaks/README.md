# `gitleaks` asdf plugin

This is the `gitleaks` integration for the [universal-asdf-plugin](../../) project.

[Gitleaks](https://github.com/gitleaks/gitleaks) is a SAST tool for detecting hard-coded secrets in git repositories. It scans commits, branches, and pull requests using built-in or custom rules to help prevent credential leaks in your codebase.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all gitleaks

# Install latest version
universal-asdf-plugin install gitleaks latest
```
