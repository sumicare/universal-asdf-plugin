# `protoc` asdf plugin

This is the `protoc` integration for the [universal-asdf-plugin](../../) project.

[protoc](https://protobuf.dev/) is the Protocol Buffers compiler for generating code from .proto definition files. It reads schema files and produces language-specific stubs that your applications use to serialize and deserialize structured data.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all protoc

# Install latest version
universal-asdf-plugin install protoc latest
```
