# `go` asdf plugin

This is the `go` integration for the [universal-asdf-plugin](../../) project.

[Go](https://go.dev/) is an open source programming language for building simple, reliable, and efficient software. It emphasizes fast compilation, concurrency primitives, and a standard toolchain that make it well-suited for cloud services, CLIs, and distributed systems.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all go

# Install latest version
universal-asdf-plugin install go latest
```
