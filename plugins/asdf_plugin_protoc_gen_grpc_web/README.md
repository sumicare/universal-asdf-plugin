# `protoc-gen-grpc-web` asdf plugin

This is the `protoc-gen-grpc-web` integration for the [universal-asdf-plugin](../../) project.

[protoc-gen-grpc-web](https://github.com/grpc/grpc-web) is the JavaScript/TypeScript code generator plugin for gRPC-Web. It creates client stubs that allow browser and frontend applications to communicate with gRPC services over HTTP/1.1 using the gRPC-Web protocol.

## Usage

```bash
# List available versions
universal-asdf-plugin list-all protoc-gen-grpc-web

# Install latest version
universal-asdf-plugin install protoc-gen-grpc-web latest
```
